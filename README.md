# 机械臂仿真 Demo

基于 Three.js + TypeScript 的机械臂仿真项目，用于demo展示与学习机器人仿真相关知识。

![机械臂仿真效果预览](./preview.png)

## 技术栈

- **前端**: Three.js + TypeScript
- **物理引擎**: Rapier3D
- **UI控制**: Tweakpane
- **动画**: GSAP
- **性能监控**: Stats.js + WebGLInfo (`renderer.info`)
- **模型**: Blender + [Sketchfab](https://skfb.ly/oOSqr)
